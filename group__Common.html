Given a bin width, a maximum area, and a series of data values, this functions automatically setups up the RooRealVar var such that:

  • The range of variable lands on a bin edge that are integer multiples of the specified bin with.
  • The range of variable minimally covers the maximum and minimum of the data points (maximum value can be over written )
